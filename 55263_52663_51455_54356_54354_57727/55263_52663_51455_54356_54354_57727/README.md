# turniej_karate
Aplikacyjny projekt zespołowy L2


## FrontEnd

### Przejdz do folderu turniej

cd ./turniej

### Zaistaluj biblioteki

npm install

### Zbuduj obecną wersję (Po każdej zmianie zbuduj ponownie)

npm run build

### Wystartuj aplikacje 

npm start

http://localhost:3000
